## 🧾 Structure
To keep everything consistent and easy to revise, every unit in the notes follows the same template:

- **Overview** — what the unit/topic is about
- **Why It Matters** — the problem it solves
- **Core Concepts** — the actual mechanics, broken down
- **Deep Dive** — internals, trade-offs, and details beyond the surface-level explanation
- **Example** — minimal working code / diagram
- **Developer Notes** — practical tips, gotchas, best practices
- **Key Takeaway** — the one-sentence summary
- **Common Mistakes** — pitfalls I want to avoid repeating
- **Quick Revision** — a one-line cheat-sheet recap

## 🧩 Core Mental Model (the one thing to always remember)
> **Agent = LLM (brain) + Tools (hands) + a loop (Thought → Action → Observation) that repeats until the goal is done.**

Every framework — smolagents, LangGraph, LlamaIndex, LangChain, CrewAI, AutoGen — is just a different
implementation of this same idea. Understand the loop once, and every new framework becomes fast to pick up.
